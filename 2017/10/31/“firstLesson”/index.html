js基本知识
1.基本概念
JavaScript 原名叫 LiveScript 是一种基于对象和事件驱动的客户端脚本语言，
最初的设计是为了检验HTML表单输入的正确性。

2.基本组成
完整的JavaScript是由3部分组成的：

  • ECMAScript (当前学习版本5.1) 规定了JS的基本语法
  • BOM 浏览器对象模型
  • DOM 文档对象模型

3.在HTML中使用JS

  • 内嵌在html当中 一般为了加载 将此标签至于body底部
  • 外链js文件

4.注意要点

    1. JS中严格区分大小写
    1. 每一条语句之后都要加上一个分号;
      5.标识符
  • 一个重要的基本概念：标识符。
  • 变量名、常量名、函数名、函数参数名、对象属性名的统称
  • 标识符的命名规则：
  • 由数字、字母、下划线和美元符($)组成
  • 首字符不能是数字
  • 不能使用JS的关键字和保留字
  • 标识符的名字可以是任意长度
    6.变量：
  • 标识符的一种，是一种变化的量，它的作用就是用来储存数据的。
  • 如何声明一个变量？使用关键字 var 变量的名字;
  • 第一个运算符：赋值运算符 “ = “ 一个等号代表赋值，而不是等于，意思是把等号右边的东西给左边；
  • 如果声明一个变量，没有初始化那么默认值是： undefined (未定义的意思)；
    7.向控制台打印内容：
  • console.log(要输出打印的内容);
    8.元素对象
  • 1.通过ID获取元素
    document.getElementById(‘id’)
  • 2.通过CSS选择器获取元素
    document.querySelector(‘css选择器’)
  • 3.元素对象也是一种数据
    var box = document.getElementById(‘box’);
    console.log(box);
    var span1 = document.getElementById(‘span1’);
  • 像getElementById 这种命名方式，叫驼峰命名法。
  • var box = document.querySelector(‘#box’);
  • var box = document.querySelector(‘.box’);
  • 只获取第一个
  • var box = document.querySelector(‘div’);
  • console.log(box);
  • var span = document.querySelector(‘div + span’);
    9.书写JS的基本流程：
  • 1 HTML + CSS 布局
  • 2 对案例进行分析（功能）
  • 3 结合已经学过的JS语法书写代码
    10.事件：文档或浏览器发生一些特定交互的瞬间
  • 点击事件 onclick
  • 鼠标移入事件 onmouseover
  • 鼠标移出事件 onmouseout
    11.初识函数：
  • 1 什么是函数?
  • 可以封装任意多条语句，代表一个功能的整体。
  • 2 声明一个函数：
  • 使用关键字 function 函数名(){ xxx…. }
  • 3 放在函数内的代码不会主动执行
  • 4 函数的调用
  • 函数名();
  • 通过事件去调用
  • 5 匿名函数
  • function (){},匿名函数不能单独存在，可以通过事件去调用
  • 6 函数和CSS里面的class非常类似，可以被复用
